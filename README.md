# Filter Database

I created this to serve as a searchable database of physical camera filters - including on/off photos and specifications of each filter. 

If you find this resource helpful and would like to support this project feel free to [buy me a coffee](https://paypal.me/chrisephoto).

[View the live site](https://chrisephoto.github.io/filter-database/)
